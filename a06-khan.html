Commands Syntax Notes
CREATE DATABASE CREATE DATABASE dbName; Used to create a new database
USE USE dbName; To use a certain database

ALTER DATABASE ALTER DATABASE dbName; To modify a database

Table Column Commands

Commands Syntax Notes
CREATE TABLE CREATE TABLE tableName ( Column1 DATATYPE, Column2 DATATYPE,... ); To create a table and its columns
RENAME TABLE RENAME TABLE oldName TO newName; To rename a table
DROP TABLE DROP TABLE tableName; To drop a table and its data
DROP COLUMN ALTER TABLE tableName DROP COLUMN columnName; To drop a column from a table

Column Attributes and Modifiers

Commands Syntax Notes
PRIMARY KEY ALTER TABLE tableName ADD CONSTRAINT PRIMARY KEY(columnName); To set a column as the primary key for a table
UNIQUE ALTER TABLE tableName ADD CONSTRAINT UNIQUE(columnName); To set a column to have unique values in a table
NOT NULL ALTER TABLE tableName MODIFY columnName DATATYPE NOT NULL; To set a column to not accept null values in a table
DEFAULT ALTER TABLE tableName ALTER columnName SET DEFAULT defaultValue; To set a default value for a column when no specific data value is specified
AUTO_INCREMENT CREATE TABLE tableName (column1 DATATYPE AUTO_INCREMENT,...); To make a column auto increment numbers, starts at 1 by default
AUTO_INCREMENT custom value ALTER TABLE tableName AUTO_INCREMENT = customValue; To make the auto increment in the table start at a custom value instead of 1 by default
FOREIGN KEY ALTER TABLE tableName ADD CONSTRAINT keyName FOREIGN KEY(columnName) REFERENCES otherTableName(columnName); To specify a column as a foreign key in a table

Table Data Commands

Commands Syntax Notes
SELECT SELECT * FROM tableName; To display all the data in a table
ADD ALTER TABLE tableName ADD columnName DATATYPE; To add a column to an existing table
INSERT INTO INSERT INTO tableName VALUES (data1, data2,...); To insert data into a table
WHERE SELECT * FROM tableName WHERE columnName = specificCriteria; To select specific data from a table
UPDATE UPDATE tableName SET columnName = newData; To update data for all rows in a certain column of a table
DELETE DELETE FROM tableName WHERE columnName = specificData To delete a row from a table
ORDER BY SELECT * FROM tableName ORDER BY last_name; To display data from a table in a specific order
