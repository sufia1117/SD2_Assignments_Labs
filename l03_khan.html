HTML Commands

Item Description
Head Tags <Head> stores the basic information of the HTML document
<Title> titles the webpage
<meta charset="utf-8"> specifies the character encoding for the document
Structural Elements <header> specifies the main heading for the webpage
<nav> creates the navigation bar for the webpage
<main> specifies the main information of the webpage
<footer> creates the footer of the webpage with copyright and contact information
Hypertext Anchor tag: <a href="">displayed text<a>
This adds a link to a webpage. Link goes into href="". target="_blank" can be added to ensure the link opens in a new tab
Horizontal Rule <hr> creates a visible line dividing the webpage horizontally
Breaks <p> holds a paragraph of text
<br> creates a line break
Formatting <b> makes words bold
<strong> also makes words bold but the style can be changed
<i> italicizes words
<emphasis> puts emphasis on a statement and italicizes the statement by default
<small> makes the statement a smaller font
Headers <h1> - <h5> creates headings, the bigger the number, the smaller the heading

CSS Styles

Styles Description
background-color Sets the background color of the webpage/table
color Sets the color of the text itself
border-collapse: collapse; Makes the table border a single line
border: 7px; border weight (thickness) in pixels
font-family text font
font-size: 7px; font size in pixels
font-weight normal or bold text choices
margin-right: auto; margin-left: auto; Centers the table
padding-left: 7px; padding-right: 7px; creates spaces in table cells

Website Design Best Practices

Avoid using too many different fonts
Avoid using too many different colors
Have lots of white space
Minimize text
Show, don't tell
